# PubWiSec: Public WiFi Security System

## Overview
PubWiSec is a Final Year Project (FYP) developed to enhance security in public WiFi environments such as cafes. The system aims to detect potential security threats and increase user awareness.

## Features
- Detection of potential Man-in-the-Middle (MITM) attacks
- Public WiFi monitoring system
- User advisory system for safer browsing

## Technologies Used
- Python (Flask)
- HTML, CSS
- Database integration

## Project Purpose
This project is developed as part of a Bachelor of Information Technology (Hons) in Cybersecurity to address the lack of security in public WiFi networks.
